Queue.io

JavaScript: Node.js,Redis, Twilio, Insomnia, Postgres, React, Bootstrap 4, Android (java), GIT, Trello, Bitbucket.

Création d'une application web « Queue.io » de gestion de file d’attente virtuelle. On a 3 acteurs : Le commerce, le client et l’employé.

Le patron de l’entreprise pourra s’enregistrer sur le site et ajouter des informations pertinentes pour aider dans la gestion de la file d’attente virtuelle : nombre maximal de clients, temps moyen estimé d’un client dans le commerce, temps d’attente du client, créer un compte d’employé, etc.

Le client pourra s’ajouter à la file d’attente et voir sa position dans la file. Pour s’ajouter, il devra écrire son nom et son numéro de téléphone. Pour accéder, le client recevra par SMS un code numérique à 4 chiffres comme vérification, ce qu’il devra rentrer. Lorsque le client le souhaite, il peut quitter la file d'attente.

L'employé est responsable de la gestion de la file d'attente via une application sur un téléphone ou une tablette. Il doit saisir son email et son mot de passe pour récupérer les informations du commerce et gérer la file d'attente. Dans cette application, l'employé pourra voir combien de personnes sont en ligne, le numéro du client qui est servi et pourra conduire au client suivant.
